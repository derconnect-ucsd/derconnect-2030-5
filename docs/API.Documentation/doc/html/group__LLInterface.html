◆ linkLayerType

SE_LLInterface_t::linkLayerType

Specifies the type of link layer interface associated with the IPInterface. Values are below.

0 = Unspecified

1 = IEEE 802.3 (Ethernet)

2 = IEEE 802.11 (WLAN)

3 = IEEE 802.15 (PAN)

4 = IEEE 1901 (PLC)

All other values reserved.
