Enumerator

isPremisesAggregationPoint 

SHALL be set if the UsagePoint is the point of delivery for a premises.

isPEV 

SHALL be set if the usage applies to an electric vehicle.

isDER 

SHALL be set if the usage applies to a distributed energy resource, capable of delivering power to the grid.

isRevenueQuality 

SHALL be set if usage was measured by a device certified as revenue quality

isSubmeter 

SHALL be set if the usage point is not a premises aggregation point
