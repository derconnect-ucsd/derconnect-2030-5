Timer* add_timer ( int  type)

Create a new timer.

When the timer expires the event type returned is the value passed to this function. This way different timers can be distinguished. is the type to be returned by event_poll, type should be a unique event type, it can be obtained by adding to the value of EVENT_NEW, e.g. add_timer (EVENT_NEW+1)

Returns
a new unarmed timer
